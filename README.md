# url-shortener
Projeto encurtador de url feito com Nodejs, express, typescript. 
Não está completo.

### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

### Instalação
`npm install` 

### Para rodar o projeto
`npm run start-dev`
